Backup copy:

Whenever you save a file, jPDFmelange creates a backup copy of the file with its previous state.
The backup file is stored in the same directory like the source file and with the extension ".bak".
If you need to restore the previous state, simply rename the backup copy <name_of_your_file.bak> to <name_of_your_file.pdf>.
