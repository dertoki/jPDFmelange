Installation

jPDFmelange is a Java program. That's why it is nearly platform independent.

This should work on Linux, Windows and MacOS:
    1. Download jPDFmelange-<version>.zip.
    2. Extracting the zip archiv creates a folder named "jPDFmelange".
    3. Start a command line.
    4. Change the current directory to jPDFmelange.
    5. Type "java -jar jPDFmelange-<version>.jar" and execute.
Windows :
A more comfortable way to start the program on Windows:
  1. Download jPDFmelange-<version>.zip. 
  2. Extracting the zip archiv creates a folder named "jPDFmelange".
  3. Use the Explorer to find jPDFmelage.bat in the new folder.
  4. Use the Explorer to create a link to jPDFmelage.bat on your desktop.
  5. Execute jPDFmelage with a double click on the link.
Linux:
More easy with almost all current Linux distributions:
  1. Download jPDFmelange-<version>.zip. 
  2. Extracting the zip archiv creates a folder named "jPDFmelange".
  3. Find jPDFmelage-<version>.jar in the new folder.
  4. Create a link to jPDFmelage-<version>.jar on your desktop.
  5. Execute jPDFmelage with a click on that link.
