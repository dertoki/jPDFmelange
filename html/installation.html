Installation

jPDFmelange is a Java program. That's why it is nearly platform independet.
This should work on Linux, Windows and MacOS:
  1. Download jPDFmelange.jar.
  2. Start a command line.
  3. Type "java -jar jPDFmelange.jar" and execute.
Windows :
A more comfortable way to start the program on Windows:
  1. Download jPDFmelange.jar and jPDFmelange.bat. 
  2. Create a new folder C:\programms\jPDFmelage and copy both files into it.
  3. Create a link to jPDFmelage.bat on your desktop.
  4. Execute jPDFmelage with a double click on the link.
Linux:
A more comfortable way to start the program on Linux:
  1. Download jPDFmelange.jar and jPDFmelange.sh. 
  2. Create a new folder ~/jPDFmelage and copy both files into it.
  3. Create a link to jPDFmelage.sh on your desktop.
  4. Execute jPDFmelage with a double click on the link.
