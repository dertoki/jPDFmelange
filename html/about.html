A program to modify existing PDF documents in a simple and user friendly way.

jPDFmelange features:

  • Merging pages from a target PDF document to the source PDF document.
  • Erasing pages from a PDF document.
  • Changing the page order of a PDF document.
  • Changing the orientation of pages in a PDF document by rotating in 90° steps.
The aim of jPDFmelange is a simple and user friendly GUI but not a wide range of functionality.

The current version of jPDFmelange is beta software. Don't use it in a production environment.
Always make a backup of your work before using it.
